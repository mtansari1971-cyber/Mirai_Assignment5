# 🎮 AI Multi-Modal Visual Novel

An interactive **Choose Your Own Adventure** web application built using **Streamlit**, **Google Gemini AI**, **Pollinations AI**, and **Google Text-to-Speech (gTTS)**.

This project was developed as the **Capstone Mini-Project** for the **MirAI School of Technology AI Internship**.

---

##  Features

-  AI-generated interactive story
-  Story Genre selection
-  Art Style selection
-  Structured JSON response from Gemini
-  AI-generated images using Pollinations AI
-  AI narration using Google Text-to-Speech (gTTS)
-  Dynamic choice buttons generated by AI
-  Stateful story progression using Streamlit Session State
-  Graceful error handling using `try...except`
-  Cached Gemini client using `@st.cache_resource`

##  How It Works

1. Select a **Story Genre**.
2. Select an **Art Style**.
3. Click **Start Story**.
4. Gemini generates a structured JSON response containing:
   - Story narration
   - Image generation prompt
   - Multiple story choices
5. Pollinations AI generates an illustration based on the image prompt.
6. gTTS converts the story narration into audio.
7. The user selects one of the AI-generated choices.
8. The story continues based on the selected option.

---

##  JSON Response Format

```json
{
  "story_text": "You wake up inside an ancient castle.",
  "image_prompt": "Anime style ancient castle interior with magical lighting",
  "options": [
    "Explore the hallway",
    "Open the mysterious door",
    "Look through the window"
  ]
}
